# 網站介紹
多數官方的導覽網站風格較為單調，即使地區導覽資訊都有包含在裡面，卻略顯無趣。為了改變這種狀況，我們設計了一個以梅山太平村為主題的介紹網站，希望以互動的方式，讓使用者來到網站內能夠在探索的過程中了解當地的特色。
